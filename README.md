# Parkinson's Drawing Classifier

A deep learning web app that detects early signs of Parkinson's disease by analyzing spiral and wave drawings.

## How it works
1. Upload a spiral drawing
2. Upload a wave drawing
3. Click Analyze
4. Get instant risk assessment from two ResNet18 models
